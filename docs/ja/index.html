KiiLib-TypeScript(JavaScript) ドキュメント

Github リポジトリへ

KiiLib-TypeScriptの簡単なドキュメントです。ビルド後のJavaScriptをそのまま使用する例もあわせて紹介します。

  • 準備

    プラットフォームに応じたJavaScriptファイルを読み込みます。現在サポートしているのは次の3つです。

    • jQuery(Webアプリ向け)
    • Titanium(スマフォアプリ開発環境)
    • Google Apps Script(Googleスプレッドシート等でマクロとして)

  • データ

    本ライブラリで登場するデータについて解説します

    • KiiUser
    • KiiGroup
    • KiiThing
    • KiiBucket
    • KiiObject
    • KiiTopic

  • App API

    アプリ全般の操作を提供するAppAPIの使い方を解説します。

    • ログイン
    • 管理者権限でログイン

  • Bucket API

    KiiBucketに関する操作を提供するBucketAPIの使い方を解説します。

    • バケツ内を検索
